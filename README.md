# Mac-Converter

Running
-------

python mac-converter.py

1. Enter/Paste Mac Addresses                                                  |
2. Ctrl-Z + Enter (windows) to convert.                                       |
3. MAC's compatible: XX:XX:XX:XX:XX:XX, XX-XX-XX-XX-XX-XX, XXXX.XXXX.XXXX

Requirements
------------

* Python 3
* Windows, should work in Linux and OSX
