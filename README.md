# Mac-Converter
Converts Mac Addresses to XX:XX:XX:XX:XX:XX, XX-XX-XX-XX-XX-XX, XXXX.XXXX.XXXX format

Running
-------

python mac_converter.py

1. Enter/Paste Mac Addresses
2. MAC's compatible: XX:XX:XX:XX:XX:XX, XX-XX-XX-XX-XX-XX, XXXX.XXXX.XXXX

Requirements
------------

* Python 3
